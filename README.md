# National-Universityy
#university-record-system-ERD

This project contains the Entity Relationship Diagram (ERD/EERD) for a centralized National University Record System designed to manage academic data of universities across Pakistan.

The system includes structured information about universities, campuses, departments, students, faculty members, degree programs, courses, admissions, examinations, and results. The ERD represents how these entities are connected and how data flows within the system.

This project helps in understanding database design concepts such as entity relationships, normalization, and enhanced ERD modeling using Lucidchart.
